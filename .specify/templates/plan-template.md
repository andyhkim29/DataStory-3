## Constitution Check

*GATE: Must pass before Phase 0 research. Re-check after Phase 1 design.*

- **Data provenance**: Source, license/access constraints, raw/processed file
  boundaries, and dataset version or download date are identified.
- **Popularity/target definition**: The analysis target is defined before
  modeling or comparison, and any proxy metric is labeled as a proxy.
- **Evidence discipline**: Planned claims map to statistics, charts, tables, or
  model results; causal claims require explicit justification or are avoided.
- **Reproducibility**: Dependencies, execution order, deterministic sampling, and
  generated artifact paths are documented.
- **Scale handling**: The plan accounts for large-data processing limits without
  silently dropping records.
- **Storytelling and caveats**: Final outputs include plain-language findings,
  labeled visuals, limitations, and alternative explanations.
- **README deliverable**: The plan includes `README.md` sections for project
  metadata, overview, data links, running instructions, contributing or open
  questions, acknowledgements, and citations.
- **Data access rule**: Data under 10 MB is included when legally allowed; larger
  data is externally hosted and downloaded or imported by the notebook/code.
- **Video scope**: A YouTube/video link is not required unless explicitly
  restored by the feature specification.
